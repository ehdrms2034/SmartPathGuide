# SmartPathGuide
Graduation Assignment of Pusan National University at 2019 late

## Description
#### 시맨틱 궤적 빅 데이터 (semantic trajectory big data) 분석 알고리즘 및 가시화
##### 목적: 과학관 방문객의 행동 파악을 위해서
##### 주제:
 1. 공통 방문 경로 추출
 2. Hot spot 탐지
 3. 방문객 분류

##### 알고리즘 구현 및 웹 기반 가시화 구현
##### 수요기업:  국내과학관,  맵 서비스 제공 업체(네이버, 카카오)
